# 10secondsデッキバトル

3ターンだけで決着する、CPU対戦の超短期カードバトルです。

## 遊び方

1. `index.html` をブラウザで開く
2. 5枚のスキルから3枚を選ぶ
3. 各ターンで残りの手札から1枚を選ぶ
4. CPUと同時公開でバトルが進む
5. 3ターン後、またはどちらかのHPが0になった時点で勝敗決定

## 現在のスキル

- 攻撃: 3ダメージ
- 防御: そのターンの被ダメージを0
- 溜め: 次の攻撃系スキルに+2
- 強攻撃: 5ダメージ、防御で無効、カウンターされると反動5ダメージ
- カウンター: 相手が攻撃系を選んでいたら3ダメージ

## 今回の実装範囲

- CPU対戦
- 5枚から3枚選ぶデッキ選択
- 3ターンの同時選択バトル
- バトルログと即リトライ導線
