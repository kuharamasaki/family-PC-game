# 10secondsデッキバトル

3ターンだけで決着する、CPU対戦の超短期カードバトルです。

## 遊び方

1. `index.html` をブラウザで開く
2. 5枚のスキルから1ターン目に使う1枚を選ぶ
3. CPUと同時公開でそのターンのバトルを行う
4. 残り4枚、残り3枚から次のターンの1枚を選ぶ
5. 3ターン後、またはどちらかのHPが0になった時点で勝敗決定

## 現在のスキル

- 攻撃: 3ダメージ
- 防御: そのターンの被ダメージを0
- 溜め: 次の攻撃系スキルに+2
- 強攻撃: 5ダメージ、防御で無効、カウンターされると反動5ダメージ
- カウンター: 相手が攻撃系を選んでいたら3ダメージ

## 今回の実装範囲

- CPU対戦
- 毎ターン、残りスキルから1枚を選ぶ進行
- 3ターンの同時選択バトル
- バトルログと即リトライ導線
